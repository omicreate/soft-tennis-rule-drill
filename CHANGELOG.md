# 更新履歴

## v1.0.4 - 2026-06-30

- ターゲットを「ソフトテニスを始めたばかりの子どもと保護者」に統一しました。
- GitHub Pagesのメタ説明、PWA manifest、README、利用条件、プライバシー方針、セキュリティ方針、docs配下の説明を更新しました。
- 共有画像とドリル画面の上部イラストを、コート図やボール中心ではなく、ルールカード中心のデザインへ差し替えました。

## v1.0.3 - 2026-06-30

- 試験モードを削除し、画面を `ドリル`、`振り返り`、`記録` の3タブに整理しました。
- 上部ステータスの `満点` を `連続` に変更しました。

## v1.0.2 - 2026-06-30

- 画面上の出典ランク表示をなくしました。
- コイントスとヒートルールの出題量を、初心者向けに少なめへ調整しました。
- 当日の大会要項・審判委員の指示を優先する注意書きを追加しました。

## v1.0.1 - 2026-06-24

- コイントスカテゴリの一部に、ルール学習ではなくアプリの出典管理・データ管理を問う設問が混ざっていたため、実際の試合前確認や審判進行に関係する内容へ差し替えました。
- Service Workerキャッシュ名を `v1.0.1` に更新しました。

## v1.0.0 - 2026-06-24

- アプリ名を `ソフテニルールドリル` に決定しました。
- 150問の4択問題、学習画面、クイズ、記録機能を公開準備版として整理しました。
- README、利用条件、プライバシー方針、セキュリティ方針、ライセンス、公開運用メモを追加しました。
- GitHub PagesとSNS共有に向けて、OGP/Twitterカードのメタ情報と共有画像を追加しました。
