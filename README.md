# ソフテニルールドリル

![ソフテニルールドリル](assets/og-image.png)

小学生・中学生、ソフトテニス初心者、審判試験を受ける人向けのスマホ用4択学習アプリです。

### 今すぐ試す: https://omicreate.github.io/soft-tennis-rule-drill/

インストール不要。スマホのブラウザで開けばそのまま使えます（ホーム画面に追加するとアプリのように起動できます）。

現在の表示バージョン: `v1.0.1` ／ 更新内容は `CHANGELOG.md` に記録しています。

## できること

- 150問の4択クイズをシャッフル出題
- 公式用語、初心者向け解説、出典ランク、確認日を表示
- 模擬試験25問と学習記録
- 間違えた問題の復習
- 2026年コイントス運用の確認
- 暑さ対策（ヒートルール）の確認
- オフラインでも開けるPWA（ホーム追加用アイコン・favicon付き）

## ルール確認方針

このアプリは公式認定アプリではありません。表示は「公式資料・連盟資料を確認して作成」に留め、公式ロゴや公式認定を思わせる表示は使いません。

出典ランク:

- `A`: JSTA公式サイト、JSTA公式PDF、JSTA公式トピックス
- `B`: 都道府県連盟、地区連盟、学連、中体連/高体連等の公開資料
- `C`: メーカー・メディア等の補助情報。問題作成の根拠には使わない

## デザイン方針

- ボールは無地の丸だけを使う
- 硬式テニス風の曲線入りボール、フェルト質感、テニスボール絵文字は使わない
- ソフトテニスのコート、審判、採点票、コールを中心に表現する
