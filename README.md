# ソフテニルールドリル

![ソフテニルールドリル](assets/og-image.png)

ソフトテニスを始めたばかりの子どもと保護者向けに、基本ルールを4択で気軽に確認できるスマホ・タブレット向けドリルアプリです。

## 公開中のアプリ

https://omicreate.github.io/soft-tennis-rule-drill/

インストール不要です。スマホやタブレットのブラウザで開けばそのまま使えます。ホーム画面に追加すると、アプリのように起動できます。

現在の表示バージョン: `v1.0.5`

## このアプリについて

- 対象: ソフトテニスを始めたばかりの子ども、保護者、初めてルールを確認する人
- 内容: 基本ルール、審判の声、試合の進め方を4択ドリルで確認
- 形式: 1セット10問の短いドリル
- 記録: 回答数、正答率、連続正解、振り返りを端末内に保存

このアプリは公式認定アプリではありません。基本的なルール練習としてできるだけ正確に作っていますが、大会や当日のルールで扱いが変わることがあります。実際の試合では、当日の大会要項、審判委員、競技上の注意の指示を優先してください。

## できること

- 150問の4択ドリルをシャッフル出題
- 1セット10問で、短い時間でも終わりやすい
- 150問をひと回りするまでは、通常ドリル内で同じ問題をできるだけ繰り返さない
- カテゴリが偏りすぎないようにランダム出題
- 公式用語を、子どもと保護者が読みやすい短い説明で表示
- 間違えた問題を振り返りで確認
- 記録画面で前向きなコメントと学習状況を表示
- オフラインでも開けるPWA

試合前のトスや暑さ対策など、年度や大会で扱いが変わりやすい内容は、基本確認として少なめに出題しています。

## 使い方

1. 公開URLをスマホやタブレットで開く
2. `ドリル` で10問に答える
3. 終わったら `振り返り` で間違えた問題を確認する
4. `記録` で回答数や正答率を見る

学習記録は、使っているブラウザの中に保存されます。別の端末や別のブラウザには自動で引き継がれません。

## プライバシー

このアプリは、氏名、学校名、メールアドレスなどを入力する機能を持ちません。学習データは主にブラウザの `localStorage` に保存され、外部サーバーへ自動送信しません。

詳しくは [PRIVACY.md](PRIVACY.md) を確認してください。

## デザイン方針

- ボールは無地の丸だけを使う
- 硬式テニス風の曲線入りボール、フェルト質感、テニスボール絵文字は使わない
- 共有画像やドリル画面では、コート図を主役にせず、ルールカード、スコア、4択の見た目を中心に表現する
