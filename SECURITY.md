# セキュリティ方針

最終更新: 2026-06-24

## 対象

この文書は、ソフテニルールドリルの公開Webアプリと、このリポジトリ内のコード、ドキュメント、問題データを対象にします。

## 基本方針

- 学習データは端末内保存を基本とする
- 外部APIへ学習データを自動送信しない
- 公開リポジトリにAPIキー、トークン、Cookie、認証情報を置かない
- 公式資料の文章や図表をそのまま転載しない
- HTML表示では必要な文字列をエスケープする
- Content Security Policyで外部スクリプト読み込みを制限する
- 公式認定、公式監修、公式ロゴ使用と誤認される表示を避ける

## 脆弱性や誤りを見つけた場合

現時点では個人開発・テスト公開のため、GitHub Issuesで詳細な個人情報や悪用手順を公開しないでください。

連絡時は、以下だけを簡潔に共有してください。

- どの画面で起きたか
- どのブラウザ、端末で起きたか
- 期待した動作と実際の動作
- 個人名、学校名、大会名を含まない再現手順
- ルール表現の誤りの場合は、確認した公式資料や大会資料

## 公開してはいけない情報

- APIキー、トークン、Cookie、認証情報
- 個人端末のバックアップファイル
- 公式資料の有料・非公開PDF
- 許可のない公式ロゴ、写真、図表
- 個人名、学校名、大会名を含む検証データ

## 依存関係

本アプリは静的HTML/CSS/JavaScriptで動作します。

最終リリース前に `npm test` を実行し、構文確認と問題データ検証を行います。

## 免責

このアプリはルール学習と審判練習を補助するものです。完全な正確性、最新性、試験合格、大会での判定結果を保証するものではありません。
