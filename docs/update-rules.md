# ルール更新確認手順

確認頻度: 3か月ごと、または大会年度の切り替わり前。

## 優先確認先

1. JSTA公式サイトの審判・技術等級ページ  
   https://www.jsta.or.jp/judge_tech
2. JSTA公式トピックス  
   https://www.jsta.or.jp/topics/
3. 2026年コイントス導入の公式案内  
   https://www.jsta.or.jp/topics/2026/04/19569.html
4. 都道府県連盟、地区連盟、中体連/高体連、学連の大会要項・競技上の注意

## 更新時に見ること

- 競技規則、審判規則、ジュニア審判マニュアルの版
- コイントス運用の適用範囲
- ラケット公認マーク廃止に関連する大会上の注意
- サービス、レット、ノーカウント、失ポイント、タイム、警告、採点票の説明変更
- ソフトテニスを始めたばかりの子どもや保護者にも伝わる表現

## データ更新

- `sources.js` に出典を追加または更新する
- `questions.js` の `sourceRefs`, `sourceRank`, `effectiveFrom`, `scopeNote`, `lastVerified` を更新する
- 出典が古い、範囲が限定される、または確認が取れない問題は `reviewStatus: "needs-review"` にする
- `npm test` を実行し、出題対象がすべて検証を通ることを確認する

## 注意

公式資料の文章や図表は転載しません。問題文と解説は、自作の短い表現に変換します。
