# リリース前確認

最終更新: 2026-06-30
対象バージョン: v1.0.4

## 結論

現時点では、GitHub Pagesで無料公開し、テストユーザーからフィードバックを受ける段階としてリリース可能。

SNSで広く案内する前に、スマホ実機で表示、ホーム追加、オフライン起動、OGP表示を確認する。

## 確認済み

- 150問の問題データがある
- 公式資料、連盟資料の出典ID、内部出典ランク、確認日を持っている
- `npm test` で構文確認と問題データ検証を行う
- `index.html` にCSPを設定している
- 学習データは `localStorage` に保存し、外部サーバーへ自動送信しない
- README、TERMS、PRIVACY、SECURITY、LICENSE、CHANGELOGを用意した
- 公式認定アプリではない旨をREADMEに記載している

## 公開前に見ること

- GitHub Pagesの初回表示
- iPhone SafariまたはChromeでの表示
- Android Chromeでの表示
- ホーム画面追加時のアプリ名とアイコン
- OGP画像、タイトル、説明文の表示
- 出典リンクの遷移
- `docs/update-rules.md` に沿った出典確認

## 残課題

- 公式資料の年度更新に合わせた問題更新
- 実際の初心者の子ども、保護者、指導者からの表現フィードバック
- スマホ・タブレット実機での継続的な表示確認
- README用スクリーンショットの追加
